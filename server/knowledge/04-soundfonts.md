# GM Soundfonts & Built-in Synths

Use GM soundfonts with `.sound("gm_name")` or `s("gm_name")`. They respond to `note()` for pitch.

**IMPORTANT: Use underscore format — `gm_epiano1` NOT `gm-epiano1`.**

## Built-in Synths

These are waveform-based synthesizers, not sample-based. Use with `note()`:

| Name | Description |
|------|-------------|
| `sine` | Pure sine wave |
| `sawtooth` | Sawtooth wave (bright, buzzy) |
| `square` | Square wave (hollow, reedy) |
| `triangle` | Triangle wave (soft, mellow) |
| `fm` | FM synthesis (use `.fmi()` and `.fmh()` for timbre) |

```js
// FM synth example
note("c3 e3 g3 b3").sound("fm").fmi(2).fmh(1.5).release(0.3)

// Sawtooth bass
note("c2 c2 e2 g2").sound("sawtooth").lpf(400).release(0.1)
```

## Usage Examples

```js
// Piano chord progression
note("<C^7 Dm7 G7 Am7>").sound("gm_epiano1").voicing().room(0.3)

// Orchestral strings
note("<C3 E3 G3 C4>").sound("gm_string_ensemble_1").release(1).room(0.5)

// Bass line
note("c2 ~ e2 g2").sound("gm_synth_bass_2").lpf(600).gain(0.8)
```
