AI Rack

AI-powered live coding music environment

Strudel patterns + Hydra visuals + Claude AI chat in one interface

What It Does

>_

Code REPL

Live music coding with Strudel, the JavaScript port of TidalCycles. Write patterns, hit Cmd+Enter, hear them instantly.

◆

Visualization

GPU shader visuals with Hydra, driven by audio analysis. Canvas 2D event lanes. Beat-reactive UI with CRT aesthetic.

⚛

AI Chat

An AI collaborator that generates, transforms, and evolves musical patterns. Ask it to change mood, genre, or energy.

Getting Started

Prerequisites

  • Node.js 20+ — required for both frontend and backend
  • ANTHROPIC_API_KEY — set as an environment variable for AI chat features

Installation

# Clone and install dependencies
npm install

Running

# Development mode — Vite on :5173, Express on :3001
npm run dev

# Or run with Electron desktop shell
npm run dev:electron

How to Use

REPL Panel

Write Strudel code in the editor. Press Cmd+Enter (or Ctrl+Enter) to evaluate. Music hot-swaps in under 50ms — no restarts, no gaps.

Chat Panel

Ask the AI to generate patterns, change mood, adjust energy, or switch genres. It knows about local samples and the built-in music engine, and injects patterns directly into the REPL.

Viz Panel

Toggle between two visualization modes:

  • Events — Canvas 2D lane visualization driven by Strudel's .onTrigger events
  • Hydra — GPU shader presets selectable from a dropdown. Audio-reactive via window.audio globals.

Voice Input

Use the microphone button for speech-to-text commands. Speak naturally to direct the AI — "make it darker", "add some hi-hats", "switch to trip hop".

AI Music Engine

The built-in music engine gives the AI deep knowledge of music theory and pattern generation.

Pattern Generation

9 genre templates — techno, house, dnb, trap, breakbeat, ambient, boom_bap, trip_hop, experimental. Each genre defines drum patterns, bass behavior, melodic tendencies, and structural conventions.

Music Theory

17 scales, 9 chord progression styles, euclidean rhythms, and arpeggios. The engine understands key signatures, voice leading, and idiomatic patterns for each genre.

Transforms

7 moods — dark, euphoric, melancholic, aggressive, dreamy, peaceful, energetic. Energy levels from 0–10. Refinements that evolve a pattern while preserving its identity.

REST API

All engine capabilities are available at /api/engine/* endpoints. Generate patterns, apply transforms, and query music theory programmatically.

Key Data Flows

  • Chat → AI → generate_pattern tool → pattern hot-swap (<50ms)
  • Audio → Meyda → audioStore → CSS vars → beat-reactive UI
  • Audio → window.audio globals → Hydra shader reactivity
  • VizEvents from Strudel .onTrigger → Canvas 2D lane visualization
